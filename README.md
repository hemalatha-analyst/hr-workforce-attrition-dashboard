# HR Workforce Insights & Employee Attrition Predictor Dashboard

## 📊 Project Overview
The **HR Workforce Insights & Employee Attrition Predictor Dashboard** is a Power BI analytics project designed to help HR teams and business leaders understand workforce composition, employee performance, and attrition behavior.

This dashboard provides actionable insights into employee retention patterns and helps organizations make data-driven decisions to reduce attrition and improve workforce planning.

---

## 🎯 Objective
- Analyze employee demographics and workforce distribution  
- Identify key drivers of employee attrition  
- Track job satisfaction, work-life balance, and performance trends  
- Detect high-risk employees likely to leave the organization  
- Support HR decision-making with predictive insights  

---

## 🧰 Tools Used
- Power BI Desktop  
- Microsoft Excel (Dataset)  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  

---

## 📁 Dataset Description
The dataset includes 100+ employee records with the following fields:
- Employee ID, Name, Age, Gender  
- Department, Job Role, Education Level  
- Salary, Performance Rating, Training Hours  
- Job Satisfaction, Work-Life Balance  
- Overtime Status, Work Mode  
- Attrition Status (Yes/No)  
- Attrition Risk Level (Low/Medium/High)  

---

## 🧹 Power Query Transformations
- Removed null and duplicate records  
- Changed data types (Date, Number, Text)  
- Created Age Groups (Young, Mid, Senior)  
- Created Experience Bands (0–2, 3–5, 5+)  
- Extracted Year & Month from Joining Date  
- Standardized categorical values  

---

## 📊 Dashboard Visuals
- KPI Cards (Employees, Attrition, Salary, Satisfaction)

## 📌 Key Insights
- Certain departments show higher attrition rates  
- Employees with low satisfaction are more likely to leave  
- Overtime increases attrition risk significantly  
- Work-life balance is a strong retention factor  
- High-risk employees can be identified using combined metrics
